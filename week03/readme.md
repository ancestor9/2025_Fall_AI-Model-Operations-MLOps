### 2. MVC (Model-View-Controller) 패턴이란?
- 소프트웨어 공학에서 사용되는 **모델-뷰-컨트롤러(Model-View-Controller)**라는 디자인 패턴
- 애플리케이션의 데이터(모델), 사용자 인터페이스(뷰), 그리고 데이터와 뷰 사이의 논리 및 흐름을 제어하는(컨트롤러) 세 부분으로 나누는 방법
